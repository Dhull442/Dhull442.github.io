# LightControlledObjectDetection ( December,2018 - present )
* Creating a dataset for object detection by varying both pose and illumination in visible
as well as infrared spectrum
* Collecting the data of objects belonging to the coco dataset using Intel RealSense R200
camera as well as by rendering the objects in Blender
* This dataset is used to characterize object detection by fine-tuning the standard object
detection networks for robust object detection integrating dark conditions as well
